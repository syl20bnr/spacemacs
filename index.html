Spacemacs is a new way to experience Emacs -- a sophisticated and polished set-up focused on

ergonomics, mnemonics and consistency.


Mnemonic

Spacemacs organizes key bindings by mnemonic namespaces as much as possible.

Discoverable

Spacemacs comes with a dedicated spacemacs-mode. Its goal is to give useful feedback and simplify maintenance tasks.

Consistent

Similar functionalities should have the same key bindings no matter which major mode is currently active.

Crowd-Configured

By using very light configuration layers which are easy to understand, Spacemacs makes it simple to contribute additional support.
