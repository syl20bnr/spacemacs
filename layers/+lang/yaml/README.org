* Description
This layer provides syntax highlighting for YAML files.

* Install
To use this contribution add it to your =~/.spacemacs=

#+BEGIN_SRC emacs-lisp
  (setq-default dotspacemacs-configuration-layers '(yaml))
#+END_SRC
