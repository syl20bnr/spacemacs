* Description
A powerline theme modeled after the vim powerline theme.

* Install

To use this contribution add it to your =~/.spacemacs=

#+BEGIN_SRC emacs-lisp
  (setq-default dotspacemacs-configuration-layers '(vim-powerline))
#+END_SRC
