* Description
This layer configures mostly Emacs built-in packages to given them better
defaults.

** Features:
- Configures packages:
  - abbrev
  - archive-mode
  - bookmark
  - conf-mode
  - cus-edit
  - dired
  - dired-x
  - display-line-numbers (only in Emacs 26.x and newer)
  - electric-indent-mode
  - easypg
  - ediff
  - eldoc
  - help-fns+ (only in Emacs 27.x and older)
  - hi-lock
  - image-mode
  - imenu
  - occur-mode
  - package-menu
  - page-break-lines
  - process-menu
  - quickrun
  - recentf
  - savehist
  - saveplace
  - subword
  - tar-mode
  - uniquify
  - url
  - visual-line-mode
  - whitespace
  - winner
  - zone
