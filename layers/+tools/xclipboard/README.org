* Description
This layer adds copy/paste support to the X-clipboard from the terminal.

* Key Bindings

| Key Binding | Description                                 |
|-------------+---------------------------------------------|
| ~SPC x p~   | Paste clipboard contents at cursor position |
| ~SPC x y~   | Copy selection to clipboard                 |
